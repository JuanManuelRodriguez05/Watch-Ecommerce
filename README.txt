- Ecommerce E-Watches

Este es un proyecto de ecommerce de relojes desarrollado en React.js como parte del aprendizaje de React.js.
El objetivo de este proyecto es construir una aplicación web básica que permita a los usuarios ver una lista de relojes, agregarlos al carrito de compras y realizar pedidos.

* Características

- Visualización de una lista de relojes disponibles.
- Funcionalidad de agregar relojes al carrito de compras.
- Realización de pedidos con envío de datos de contacto.
- Uso de React Router DOM para la navegación entre diferentes páginas.
- Integración de la librería Toastify para mostrar notificaciones en la aplicación.

* Tecnologías utilizadas

- React.js
- React Router DOM
- CSS
- Toastify
